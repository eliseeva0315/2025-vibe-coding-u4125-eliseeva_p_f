Для улучшения был использован следующий промпт:

```text
Добавь в бота кнопку «⬅️ Назад».

Хочу, чтобы она работала в любом диалоге (wishlist, добавление сотрудника, выбор сотрудников):
— при нажатии очищается текущий диалог
— бот возвращает в главное меню
— показывает MAIN_MENU_KEYBOARD

Также добавь эту кнопку в сообщения, где бот просит ввести данные.

Ничего больше не меняй, только аккуратно добавь эту кнопку.
```

В результате:
- добавлена кнопка «⬅️ Назад»  
- реализован быстрый выход из любого сценария  
- улучшена навигация
